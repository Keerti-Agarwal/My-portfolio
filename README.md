# Portfolio Website

A personal portfolio website to showcase projects and skills, featuring smooth UI animations and fully responsive design.

## Features

- **Typewriter animation** cycling through developer roles
- **Smooth scroll navigation** with active link highlighting
- **Responsive hamburger menu** for mobile
- **Contact form** with client-side validation
- **Project showcase** with tech stack tags

## Tech Stack

HTML5 · CSS3 · JavaScript

## How to Run

Open `index.html` in any browser.

## Project Structure

```
portfolio-website/
├── index.html   # Full single-page portfolio
├── style.css    # All styles + responsive
├── app.js       # Typewriter, scroll effects, form validation
└── README.md
```

## Sections

1. Hero — name, roles typewriter, CTA buttons
2. About — background, education, quick facts
3. Skills — tag grid + animated progress bars
4. Projects — 3 project cards with details
5. Contact — info + validated contact form
